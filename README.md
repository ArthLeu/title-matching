## Useful Scripts
Installing required packages
``` bash
pip3 install -r requirements.txt
```

Store train data at location:
``` bash
dataset/train/
```

Store test data at location:
``` bash
dataset/test/
```

Running the program:
``` bash
python3 main.py
```
